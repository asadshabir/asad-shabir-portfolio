## Tech Stack (this site)

- **Framework:** Vite + React 18 + TypeScript
- **Styling:** Tailwind CSS v3 + custom design tokens (HSL semantic colors)
- **Animations:** Framer Motion + custom Aceternity-style components
- **UI Kit:** shadcn/ui (Radix primitives)
- **Color scheme:** `#0A0A0A` background · `#22D3EE` neon cyan · `#C026D3` magenta

### Run locally
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```

## AI Assistant (in-app chatbot)

The floating "Ask Asad AI" assistant is **strictly scoped** to Asad's profile —
skills, projects, experience, and contact info. It returns answers grounded in
`src/lib/asadKnowledge.ts` and includes reference links (live demos / GitHub
repos) for project-related questions.

To upgrade to a real LLM-backed agent (OpenAI Agents SDK + Groq), enable
**Lovable Cloud** so a secure edge function can hold the API keys.
